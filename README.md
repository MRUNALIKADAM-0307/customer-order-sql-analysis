# 📊 Customer Order Analysis using SQL

## 📌 Overview
This project analyzes a retail order dataset consisting of 20 customer transactions across multiple cities and products. The goal was to understand customer spending patterns, product performance, and overall sales trends using SQL.

## 🛠 Tools Used
- MySQL
- SQL (JOIN, GROUP BY, HAVING, Subqueries)

## 🗂 Dataset Details
- 20 orders
- 20 customers from different cities
- 20 products with varying prices and discounts

## 🔍 Key Analysis
- Calculated total quantity sold per product
- Computed total sales per customer using JOIN operations
- Identified customers with spending greater than 50,000
- Analyzed products priced above average using subqueries

## 💡 Key Insights
- A small group of customers contributed to the highest revenue
- Certain high-priced products significantly impacted total sales
- Discounts influenced final order values

## ▶️ How to Run
1. Run `schema.sql`
2. Run `data.sql`
3. Run `analysis.sql`
