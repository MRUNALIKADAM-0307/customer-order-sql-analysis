# 📊 Customer Order Analysis using SQL

## 📌 Overview
This project analyzes customer orders, product pricing, and sales performance using SQL.

## 🛠 Tools Used
- MySQL
- SQL (Joins, Aggregations, Subqueries)

## 🔍 Key Analysis
- Total quantity ordered per product
- Total sales per customer
- High-spending customers (>50,000)
- Product pricing analysis

## 💡 Insights
- Identified top customers by spending
- Found best-selling products
- Analyzed pricing trends

## ▶️ How to Run
1. Run `schema.sql`
2. Run `data.sql`
3. Run `analysis.sql`
